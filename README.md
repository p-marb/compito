# compito
An open source, free personal task planner. Minimal design to streaminline productivity.

- Keep track of different categories of tasks you have planned and track your progress. 
- Multiple projects, customizable categories & colors, easy import + export for sharing progress.
- Fully locally hosted on your computer. All projects are stored in JSON in your browsers localStorage.
